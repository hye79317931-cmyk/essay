문제 스크린샷

문제 스크린샷PSAT 앱처럼 스크린샷 후 여기를 누르고 “문제 스샷 붙여넣기”를 누르면 문제 페이지로 추가됩니다.

문제 빈 페이지 추가 → 해당 페이지 선택 → 스샷 붙여넣기. 선택한 페이지에 그대로 들어가고 선택도 유지됩니다.

해설 / 모범답안 스크린샷

해설 스크린샷PSAT 앱처럼 스크린샷 후 여기를 누르고 “해설 스샷 붙여넣기”를 누르면 해설 페이지로 추가됩니다.

해설 빈 페이지 추가 → 해당 페이지 선택 → 스샷 붙여넣기 순서로 넣어. 채점 화면에서 페이지 넘김 가능.
