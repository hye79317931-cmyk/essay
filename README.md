# 2차 서술형 답안연습 - PSAT 기반 v6

기존 2차 앱을 이어붙이지 않고, PSAT 앱의 붙여넣기 구조를 기준으로 새로 만든 버전입니다.

## 핵심 기능
- 대분류: 형법 / 형사소송법 / 행정법 / 교정학
- 문제 스크린샷 여러 페이지 추가
- 해설/모범답안 스크린샷 여러 페이지 추가
- 문제등록 화면에서 PSAT 방식 붙여넣기
- 문제풀이 화면에서 문제 페이지 이전쪽/다음쪽
- 채점 화면에서 해설 페이지 이전/다음
- 목차연습 / 실전답안
- 복습필요 랜덤
- 목록 수정/복제/삭제
- 백업/복원
- PWA 홈화면 설치
