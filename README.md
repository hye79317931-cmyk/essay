# 2차 서술형 답안연습 v10

PSAT 객관식 앱과 별개로 처음부터 만든 2차 서술형용 PWA입니다.

## 대분류
- 형법
- 형사소송법
- 행정법
- 교정학

## 반영한 구조
- 문제 스크린샷
- 모범답안/해설 스크린샷
- 채점포인트
- 배점
- 제한시간
- 연도/회차
- 실전답안 / 목차연습
- 중단/이어풀기
- 제출 후 내 답안과 모범답안 비교
- 채점포인트 체크
- 점수·체감·복습필요·메모 저장
- 복습필요 / 점수낮음 / 시간초과 / 포인트누락 필터
- 이전 풀이시간 비교
- 백업 / 복원

## v3 변경사항
- PWA 설치용 아이콘 추가
- manifest icons 추가
- 홈화면 설치 버튼 추가
- service worker 캐시 v3로 변경

## v5 변경사항
- 문제/해설 스크린샷 즉시 붙여넣기 버튼 추가
- 해설 이미지 추가 버튼 추가
- 문제와 해설 이미지를 여러 장 저장 가능
- 문제풀이/채점 화면에서 한 페이지씩 넘겨보기 가능
- PWA 설치 버튼과 아이콘 유지

## v5 수정
- v4가 캐시/혼합 코드 때문에 반영이 안 되는 문제를 정리했습니다.
- app.js/styles.css에 캐시 무력화 쿼리를 추가했습니다.
- service worker 캐시를 v5로 올렸습니다.


## v6 긴급 수정
- `Cannot read properties of null (reading 'addEventListener')` 오류 수정
- 문제풀이 왼쪽에 이전쪽 / 다음쪽 버튼 누락 수정
- 일부 HTML/JS 캐시 불일치에도 앱이 멈추지 않도록 안전 처리
- 캐시 버전 v6 적용


## v7 수정사항
- 모바일 Chrome에서 '문제 스샷 붙여넣기' 버튼을 누르면 클립보드 권한 오류가 뜨던 문제 수정
- 버튼 클릭 시 클립보드를 강제로 읽지 않고 붙여넣기 영역에 포커스되도록 변경
- PC: 버튼 클릭 후 Ctrl+V
- 모바일: 파일 선택 / 해설 이미지 추가 사용
- 캐시 버전 v7 적용


## v8 수정사항
- PSAT 앱처럼 버튼에서 클립보드 이미지 읽기를 다시 시도하도록 변경
- 모바일 Chrome에서 클립보드 권한이 막히면 자동으로 파일 선택창을 열도록 변경
- 문제/해설 모두 동일 적용
- 캐시 버전 v8 적용


## v9 수정사항
- PSAT 앱 방식과 동일하게 붙여넣기 대상 선택 구조 적용
- 문제/해설 붙여넣기 영역에 data-paste-target 적용
- 버튼 클릭 시 navigator.clipboard.read()로 직접 이미지 읽기
- 전역 paste 이벤트 처리 추가
- 현재 붙여넣기 대상 강조 표시
- 캐시 버전 v9 적용


## v10 수정사항
- 문제/해설 영역에 `클립보드 권한 허용` 버튼 추가
- 버튼 클릭 시 브라우저의 clipboard-read 권한 요청을 명시적으로 실행
- 권한이 이미 차단된 경우 사이트 설정에서 허용하라는 안내 표시
- 캐시 버전 v10 적용
