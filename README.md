# 2차 서술형 답안연습 v5

PSAT 객관식 앱과 별개로 처음부터 만든 2차 서술형용 PWA입니다.

## 대분류
- 형법
- 형사소송법
- 행정법
- 교정학

## 반영한 구조
- 문제 스크린샷
- 모범답안/해설 스크린샷
- 채점포인트
- 배점
- 제한시간
- 연도/회차
- 실전답안 / 목차연습
- 중단/이어풀기
- 제출 후 내 답안과 모범답안 비교
- 채점포인트 체크
- 점수·체감·복습필요·메모 저장
- 복습필요 / 점수낮음 / 시간초과 / 포인트누락 필터
- 이전 풀이시간 비교
- 백업 / 복원

## v3 변경사항
- PWA 설치용 아이콘 추가
- manifest icons 추가
- 홈화면 설치 버튼 추가
- service worker 캐시 v3로 변경

## v5 변경사항
- 문제/해설 스크린샷 즉시 붙여넣기 버튼 추가
- 해설 이미지 추가 버튼 추가
- 문제와 해설 이미지를 여러 장 저장 가능
- 문제풀이/채점 화면에서 한 페이지씩 넘겨보기 가능
- PWA 설치 버튼과 아이콘 유지

## v5 수정
- v4가 캐시/혼합 코드 때문에 반영이 안 되는 문제를 정리했습니다.
- app.js/styles.css에 캐시 무력화 쿼리를 추가했습니다.
- service worker 캐시를 v5로 올렸습니다.
