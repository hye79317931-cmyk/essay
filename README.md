# 2차 서술형 답안연습 v3

PSAT 객관식 앱과 별개로 처음부터 만든 2차 서술형용 PWA입니다.

## 대분류
- 형법
- 형사소송법
- 행정법
- 교정학

## 반영한 구조
- 문제 스크린샷
- 모범답안/해설 스크린샷
- 채점포인트
- 배점
- 제한시간
- 연도/회차
- 실전답안 / 목차연습
- 중단/이어풀기
- 제출 후 내 답안과 모범답안 비교
- 채점포인트 체크
- 점수·체감·복습필요·메모 저장
- 복습필요 / 점수낮음 / 시간초과 / 포인트누락 필터
- 이전 풀이시간 비교
- 백업 / 복원
